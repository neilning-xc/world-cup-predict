## 权重设计依据

### 为什么博彩赔率权重最高（25%）？

1. **信息聚合器**: 赔率由专业机构综合所有已知信息计算，包括未公开的伤病、阵容变化等
2. **市场共识**: 数百万投注者的集体智慧，反映最广泛的市场判断
3. **实时更新**: 赔率随最新信息实时调整，时效性最强
4. **验证过的准确性**: 历史数据显示赔率预测准确率约65-70%

### 为什么阵容质量提升至15%？

- 在国家队层面，一个超级巨星（如梅西、姆巴佩）或核心中轴线球员的伤缺对球队战力的影响是毁灭性的，其重要性甚至超过了历史战绩。

### 为什么FIFA排名和近期战绩并列（各15%）？

- **FIFA排名**: 反映长期综合实力，是权威的基准参考
- **近期战绩**: 反映当前状态和士气，弥补排名的滞后性
- 两者互补：一个看"静态实力"，一个看"动态状态"

### 为什么历史交锋降至5%，预选赛保持10%？

- **历史交锋 (5%)**: 国家队比赛频率极低，很多交锋记录是4年前甚至8年前的数据，当时的球员和教练早已更迭，参考价值有限。
- **预选赛表现 (10%)**: 正式赛事比热身赛更有参考价值，能体现球队在真刀真枪比赛中的抗压能力。

### 为什么主客场、赛程、战术克制权重较低？

- **主客场（8%）**: 2026在北美举办，部分中北美及南美球队有地利和气候适应性优势，但不是绝对决定性因素
- **赛程/体能（4%）**: 影响存在但难以精确量化
- **战术克制（3%）**: 风格相克存在但主观判断成分大

## 综合评分计算

### 计算公式

```
总分 = Σ(各因素得分 × 对应权重)
```

### 示例计算 (巴西 vs 德国)

| 因素 | 巴西得分 | 德国得分 | 权重 | 巴西加权 | 德国加权 |
|------|----------|----------|------|----------|----------|
| F1 赔率 | 70 | 50 | 0.25 | 17.50 | 12.50 |
| F6 阵容 | 70 | 65 | 0.15 | 10.50 | 9.75 |
| F2 排名 | 65 | 55 | 0.15 | 9.75 | 8.25 |
| F3 近期 | 75 | 65 | 0.15 | 11.25 | 9.75 |
| F5 预选赛 | 80 | 70 | 0.10 | 8.00 | 7.00 |
| F7 主场 | 65 | 55 | 0.08 | 5.20 | 4.40 |
| F4 交锋 | 50 | 50 | 0.05 | 2.50 | 2.50 |
| F8 体能 | 70 | 70 | 0.04 | 2.80 | 2.80 |
| F9 战术 | 50 | 50 | 0.03 | 1.50 | 1.50 |
| **总计** | | | **1.00** | **69.00** | **58.45** |

**差值**: 69.00 - 58.45 = 10.55

## 预测映射规则

| 总分差 | 预测结果 | 典型比分 |
|--------|----------|----------|
| > 20 | 大胜 | 3-0, 3-1, 4-1 |
| 10-20 | 小胜 | 2-0, 2-1 |
| 5-10 | 微胜 | 1-0, 2-1 |
| 0-5 | 平局或险胜 | 1-1, 1-0, 0-0 |
| < 0 | 反向预测 | 预测低分队获胜 |

### 置信度计算

```
置信度 = |总分差| / 20 × 100%

示例: 差值10.55
置信度 = 10.55 / 20 × 100% = 52.75% → 中等置信度
```

| 置信度 | 等级 |
|--------|------|
| > 80% | 高 |
| 50-80% | 中 |
| < 50% | 低 |

## 调整建议

### 如果数据缺失

| 缺失因素 | 调整方案 |
|----------|----------|
| 赔率缺失 | 将25%权重平均分配给其他因素 |
| 排名缺失 | 使用ELO评分替代 |
| 近期战绩缺失 | 增加预选赛权重 |
| 阵容信息缺失 | 使用默认评分(60分) |

### 特殊情况处理

| 情况 | 处理方式 |
|------|----------|
| 点球大战 | 预测常规时间平局，附加点球预测 |
| 加时赛 | 预测90分钟结果，注明可能进入加时 |
| 小组赛末轮 | 考虑球队出线形势对战意的影响 |
