# 2026世界杯比赛比分预测

基于多维度数据分析的世界杯比赛比分预测工具。

## 快速开始

在对话中提到以下关键词即可触发：
- "预测今天的比赛"
- "世界杯比分预测"
- "今天有什么比赛"

## 数据获取方式

### 核心数据源

| 数据类型 | 来源 | 获取方式 | 用途 |
|----------|------|----------|------|
| 当天比赛 | FIFA官网/ESPN | WebFetch | 比赛列表 |
| 博彩赔率 | oddsportal.com | WebFetch | 市场共识 |
| FIFA排名 | fifa.com | WebFetch | 实力基准 |
| 近期战绩 | transfermarkt.com | WebFetch | 状态评估 |
| 历史交锋 | transfermarkt.com | WebFetch | 心理优势 |
| 阵容信息 | transfermarkt.com | WebFetch | 伤病影响 |
| 预选赛 | fifa.com | WebFetch | 赛事表现 |

### 数据获取流程

```
1. 获取当天比赛列表 (FIFA/ESPN)
   ↓
2. 对每场比赛并行获取6类数据
   ├── 赔率数据
   ├── 两队FIFA排名
   ├── 近10场战绩
   ├── 历史交锋记录
   ├── 阵容/伤病信息
   └── 预选赛表现
   ↓
3. AI综合分析
   ↓
4. 输出预测结果
```

## 核心权重分配

### 权重总览

| 因素 | 权重 | 说明 |
|------|------|------|
| **博彩赔率** | **25%** | 市场共识，信息密度最高 |
| **FIFA排名** | **15%** | 长期综合实力基准 |
| **近期战绩** | **15%** | 当前状态和士气 |
| **历史交锋** | **10%** | 心理优势和风格克制 |
| **预选赛表现** | **10%** | 正式赛事竞技水平 |
| **阵容质量** | **10%** | 伤病/停赛影响 |
| **主客场因素** | **8%** | 2026北美地利 |
| **赛程/体能** | **4%** | 疲劳程度 |
| **战术克制** | **3%** | 风格相克 |

### 权重设计依据

1. **博彩赔率 (25%)**: 赔率是专业机构综合所有已知信息计算得出，包含伤病、阵容等隐含信息，是最高效的信息聚合器

2. **FIFA排名 + 近期战绩 (各15%)**: 前者反映静态实力，后者反映动态状态，两者互补

3. **历史交锋/预选赛/阵容 (各10%)**: 中等权重，提供有价值的补充信息

4. **主客场/赛程/战术 (8%+4%+3%)**: 辅助因素，影响幅度较小但不可忽视

### 评分标准

| 因素 | 评分方式 |
|------|----------|
| 赔率 | 概率转换 → 胜率百分比 |
| 排名 | 排名差 → 加分/减分 |
| 近期战绩 | 胜平负 → 状态分 |
| 历史交锋 | 胜场比例 → 交锋分 |
| 预选赛 | 小组排名 → 排名分 |
| 阵容 | 球星数量 + 伤病影响 |
| 主场 | 地理位置 → 加分 |
| 体能 | 休息天数 → 体能分 |
| 战术 | 风格关系 → 加分/减分 |

## 预测结果

### 总分差 → 预测映射

| 总分差 | 预测结果 | 典型比分 |
|--------|----------|----------|
| > 20 | 大胜 | 3-0, 3-1, 4-1 |
| 10-20 | 小胜 | 2-0, 2-1 |
| 5-10 | 微胜 | 1-0, 2-1 |
| 0-5 | 平局或险胜 | 1-1, 1-0, 0-0 |
| < 0 | 反向预测 | 预测低分队获胜 |

**简洁模式**: 只显示预测比分和置信度

**详细模式**: 包含各因素分析、权重说明、关键看点
